Unverlangt zugesandte e-mail Werbung - spam

Interessante Urteile zum www: Unverlangt zugesandte e-mail Werbung: Die Rechtssprechung zu unverlangt zugesandten e-mails ist - wie es so schön heißt - uneinheitlich. Dies überrascht einigermaßen, nachdem bei unverlangt zugesandten Telefaxen bzw. Telefonanrufen ganz überwiegend davon ausgegangen wird, dass solche unzulässig ist ( Urteil BGH WM 2000, S. 1264 ). Anders soll dies nur bei Gewerbetragenden sein, bei denen von einem mutmaßlichen Einverständnis ausgegangen werden kann. Für rechtlich unzulässig halten e-mail Werbung: - Landgericht Traunstein ( NJW-CoR 1997, S. 494 ) - Amtsgericht Brake ( NJW 1998, S. 3209 ) - Landgericht Berlin ( CR 1998, S. 623 ) Für rechtlich zulässig halten unverlangt zugesandte e-mail Werbung: - Landgericht Augsburg ( NJW 2000, S. 593 ) - Landgericht Braunschweig ( NJW-CoR 2000, S. 235 )
